### Instalasi

1. Clone repository
```bash
git clone https://github.com/Whyuugl/TravelEase.git
cd TravelEase
```

2. Install dependencies
```bash
npm install
# atau
yarn install
```

3. Jalankan development server
```bash
npm run dev
# atau

```

4. Buka browser dan akses `http://localhost:5173`
